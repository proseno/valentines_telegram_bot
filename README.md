# Valentine Telegram Bot

A romantic Telegram bot that invites your Valentine and reveals the plan only after "Yes".

## Setup
1. Create a bot with @BotFather and get the token.
2. Copy `.env.example` to `.env` and paste your token.
3. Install dependencies and run the bot.

```bash
npm install
npm start
```

## Customize
Edit these strings in `src/index.js`:
- `inviteText`
- `yesRevealText`
- `wrongAnswerText`

You can also change the buttons or add more steps.
